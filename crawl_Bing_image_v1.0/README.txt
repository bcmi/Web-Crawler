How to use?

1. prepare your query list as 'queries.txt'

2. 'mt_Bing.py' is a multithead wrapper for 'Bing.py', you can run the script as follows,
   $ python mt_Bing.py
   The number of threading can be changed by setting $nThreads in 'mt_Bing.py'
   The save dir can be changed by setting $BaseSaveDir in 'Bing.py'
