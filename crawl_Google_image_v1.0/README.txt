How to use?

1. prepare your query list as 'queries.txt'

2. 'mt_Google.py' is a multithead wrapper for 'Google.py', you can run the script as follows,
   $ python mt_Google.py
   The number of threading can be changed by setting $nThreads in 'mt_Google.py'
   The save dir can be changed by setting $BaseSaveDir in 'Google.py'
