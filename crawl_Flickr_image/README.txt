How to use?

1. prepare your query list as 'queries.txt'

2. 'mt_flickr.py' is a multithead wrapper for 'flickr.py', you can run the script as follows,
   $ python mt_flickr.py
   The number of threading can be changed by setting $nThreads in 'mt_flickr.py'
   The save dir can be changed by setting $BaseSaveDir in 'flickr.py'
