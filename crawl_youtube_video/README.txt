How to use?

1. prepare your query list as 'queries.txt'

2. run $python get_url.py
   get downloadable urls

3. run $python mt_YouTube.py
   utilize Youtube API 'youtube-dl' to download videos and       descriptions

The number of threading can be changed by setting $nThreads in 'mt_YouTube.py'
   The save dir can be changed in 'get_url.py' and 'mt_YouTube.py'
